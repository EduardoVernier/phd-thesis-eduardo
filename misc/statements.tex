\begin{enumerate}

\item Transforming abstract data into a useful picture is a hard problem. Transforming it into a useful \textit{video} is even harder. 
\vspace{.3cm}

\item Visual quality and temporal stability are conflicting criteria.
\vspace{.3cm}

\item Stability does not imply absence of movement, it implies the correct amount of movement.
\vspace{.3cm}

\item Dynamic projection quality is a mental challenge for it implies reasoning about the \textit{derivative} of an already complex function that we do not fully understand -- the static projection itself.
\vspace{.3cm}

\item \textit{``You can't improve what you don't measure.''} -- Peter Drucker
\vspace{.3cm}

\item \textit{``To measure is to know.''} -- Lord Kelvin
\vspace{.3cm}

\end{enumerate}
